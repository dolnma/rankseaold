More Information

In CS:GO, there are 6 different types of crosshair settings you can choose from. (cl_crosshairstyle 0-5)
Styles 0 and 1 have very little customizability, and can be easily done in-game.
Styles 3,4,and 5 are very similar. Style 4 is the static version, which is what this generator shows.
Style 3 is a very dynamic version of style 4 (i.e. it expands when jumping/shooting/running)
Style 5 is a slightly dynamic version of style 4.

Style 2 is fairly complicated, and varies depending on what gun you're using, hence the lack of support in this generator.
However, style 2 is very similar to style 4, except with a few additions which can be made using console commands. You can make the basic static version of the crosshair here, and then make changes using your console for style 2.
The unlisted settings for style 2 are as follows: cl_crosshair_dynamic_maxdist_splitratio, cl_crosshair_dynamic_splitalpha_innermod, cl_crosshair_dynamic_splitalpha_outermod, and cl_crosshair_dynamic_splitdist.
